Markov Decision Processes - Project 4 ()

This assignment was done using the CMU reinforcement learning simulator, which can be accessed at https://www.cs.cmu.edu/~awm/rlsim/.

To run the simulator, unzip the .zip file or clone the directory on github. Then, navigate to the directory, and then run the command 'java -jar rl_sim.jar' to start the simulator. The maze files that were used for this assignment can be found in the mazes folder.
